\# Titanic Survival Prediction using Machine Learning and Power BI

\## 1. Title & Description

This project presents a complete data analytics and machine learning workflow using the Titanic dataset. It includes data cleaning, exploratory data analysis (EDA), predictive modeling, and an interactive Power BI dashboard to analyze and predict passenger survival.

\---

\## 2. Problem Statement

The objective of this project is to analyze the factors affecting passenger survival on the Titanic and build a machine learning model capable of predicting whether a passenger survived based on demographic and travel-related information.

\---

\## 3. Data Source

\* \*\*Dataset:\*\* Titanic Dataset

\* \*\*Source:\*\* Kaggle

\* \*\*Records:\*\* 891 passengers

\* \*\*Features:\*\* Passenger information including age, gender, passenger class, fare, embarkation point, and survival status.

\---

\## 4. Methodology

The project was completed in the following stages:

\* Data Collection

\* Data Cleaning and Preprocessing

\* Exploratory Data Analysis (EDA)

\* Feature Engineering

\* Machine Learning Model Development

\* Model Evaluation

\* Dashboard Development using Power BI

Python libraries used:

\* Pandas

\* NumPy

\* Matplotlib

\* Seaborn

\* Scikit-learn

\---

\## 5. Key Findings

\* Female passengers had a significantly higher survival rate.

\* First-class passengers were more likely to survive than third-class passengers.

\* Passenger age and fare influenced survival probability.

\* Missing values were successfully handled to improve data quality.

\---

\## 6. Results

\* Cleaned and processed the raw Titanic dataset.

\* Built a Decision Tree Classifier for survival prediction.

\* Evaluated model performance using accuracy.

\* Developed an interactive Power BI dashboard for data visualization and business insights.

\---

\## 7. Conclusions

This project demonstrates an end-to-end data analytics workflow from raw data preprocessing to predictive modeling and dashboard visualization. It highlights practical applications of Python, machine learning, and business intelligence tools in solving real-world analytical problems.
